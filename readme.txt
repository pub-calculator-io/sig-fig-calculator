Significant figures are essential in various situations. Here, we determine the rules of sig figs and how we can apply them in the real world.

[https://www.calculator.io/sig-fig-calculator/](https://www.calculator.io/sig-fig-calculator/)

== Usage ==

Add the Sig Fig Calculator shortcode to your page, post or sidebar:

`[ci_sig_fig_calculator]`

Add the following code to your template where you would like the Sig Fig Calculator to appear:

`<?php display_ci_sig_fig_calculator(); ?>`
